## ✨ Features

### 🎯 Part 1 - Single Pokémon Search
- Search individual Pokémon by name
- View detailed information including types and sprite
- Beautiful glassmorphic UI with animations

### 📊 Part 2 - Multiple Pokémon Search
- Search multiple Pokémon at once (comma-separated)
- Display results in an organized table
- Responsive design with hover effects

### 🔍 Part 3 - Filterable Pokédex
- Filter Pokémon by type
- Pagination support
- Real-time filtering with smooth animations
- Type-specific color coding

### 🎨 Bonus Features
- ✨ Stunning glassmorphic design
- 🎭 Type-specific icons and colors
- 🌊 Smooth animations and transitions
- 📱 Fully responsive layout
- ⚡ Optimized performance with React Query
- 🗄️ Efficient database indexing
- 🔄 Automatic caching
- 🎯 SEO optimized

## 🚀 Tech Stack

- **Frontend**: Next.js 14, TypeScript, Material UI
- **Backend**: tRPC, Next.js API Routes
- **Database**: PostgreSQL (Neon DB), Prisma ORM
- **State Management**: React Query, tRPC
- **Styling**: Material UI, Emotion, CSS-in-JS
- **Deployment**: Vercel
